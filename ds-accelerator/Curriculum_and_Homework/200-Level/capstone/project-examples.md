# Example 200 Level Projects

The following are examples of 200 level projects from the last couple of years. Don't feel like you have to build a perfect model for this project - strong projects can surround thorough exploratory data analysis.

Which attributes/dimensions that we know about a Higher Ed conversion could be significant if we want to go down the path of classifying whether or not we think somebody will apply and enroll? Or, flipping that a little, for which dimensions/attributes of a student who submits a lead form does that hypothesis about CR being an indicator of “good fit” seem to hold true, and when does it not?
