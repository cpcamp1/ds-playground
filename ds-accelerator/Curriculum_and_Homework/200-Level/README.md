## Week 9 |09/30/2024 | Supervised Learning: Regression
#### Learning Objectives
* Supervised machine learning: Linear regression and loss function for regression
* Understand differences between normal equation and gradient descent (and pros and cons)
* Understand Ridge and Lasso regression techniques
  
#### Videos
[The Complete Machine Learning Course with Python](https://redventures.udemy.com/machine-learning-course-with-python/learn/lecture/8453648#overview)

* [Multiple Regression](https://www.youtube.com/watch?v=zITIFTsivN8&list=PLblh5JKOoLUIzaEkCLIUxQFjPIlapw8nU&index=3) (5min)
* [Regularization - Ridge (L2) Regression](https://www.youtube.com/watch?v=Q81RR3yKn30) (20min)
* [Regularization - Lasso (L1) Regression](https://www.youtube.com/watch?v=NGf0voTMlcs) (8min)
* [Regularization - Elastic Net Regression](https://www.youtube.com/watch?v=1dKRdX9bfIo) (5min)
  
#### Homework
